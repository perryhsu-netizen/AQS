## 🎨 自定義指南

### 🌈 修改配色方案
在 `style.css` 中修改 CSS 變數：
```css
:root {
    /* 主色調 */
    --primary-gradient: linear-gradient(135deg, 你的顏色1, 你的顏色2);
    --secondary-gradient: linear-gradient(135deg, 你的顏色3, 你的顏色4);
    
    /* 品牌色 */
    --primary-color: #你的主色;
    --accent-color: #你的強調色;
}
```
